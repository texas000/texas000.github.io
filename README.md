# Debugging

To debug the site locally, you can use Jekyll's built-in development server.

1.  **Install dependencies:**
    ```bash
    bundle install
    npm install
    ```

2.  **Build and serve:**
    ```bash
    bundle exec jekyll serve
    ```

3.  Open your browser to `http://localhost:4000`.
